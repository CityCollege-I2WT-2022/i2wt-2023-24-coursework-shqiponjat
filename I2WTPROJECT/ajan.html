HTML TAGS

html tag - Defines the area where html will be contained.

head tag - Defines the head section of the html.

meta tag - Provides the characters of many languages and makes the design responsive.

title tag - Diplays the webpage name on the browser title bar.

link tag - Connects to external stylesheets and to add an icon to the page.

script tag -Connects to the javascript file.

body tag - Defines the body section of the html.

header tag - Defines the header section of the html.

section tag - Divides the page into sections.

div tag - Divides a section into divisions.

h2 tag - Gives a heading of level 2.

ul tag - Creates an unordered list.

li tag - Separates the elements of a list.

a tag - Hyperlinks other pages of the site.

h3 tag - Gives a heading of level 3.

button tag - Creates a button that can be clicked.
