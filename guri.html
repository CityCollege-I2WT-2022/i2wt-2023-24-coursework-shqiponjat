HTML TAGS

section tag - Used to divide the page into smaller parts (sections).

div tag - Used to divide a section into smaller parts (divisions).

p tag - Used to create paragraphs where information was written.

img tag - Used to insert images in the page.

h3 tag - Used to give a heading of level 3 (smaller than 1 and 2).

footer tag - Used to define the footer of the page.

a tag - Used to hyperlink other pages of the site and other websites

ul tag - Used to create a list that is not ordered

li tag - Used to separate the elements of the list that was created

i tag - Used to connect back to icons that were imported in the page earlier.
